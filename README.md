# MuTao

音楽再生、ロール一括付与・剥奪、絵文字保存、YouTube等のサムネの保存など、様々な機能を無心に詰め込んだBot。

ytdl-coreが原因で再生できない問題がある可能性があるため、エラーが発生した場合はnpm uninstall ytdl-coreし、node-modulesの中のytdl-coreフォルダを抹消して下さい。
